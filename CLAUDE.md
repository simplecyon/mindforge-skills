# MindForge — Claude Code Guidelines

## If You Are an AI Agent

This repo contains thinking skills. Your job is to use them appropriately.

### The Rhythm

```
模糊 → 清晰 → 发散 → 深入 → 质疑 → 收拢
clarifier → muse → thinker → janus → conductor
```

### Your Responsibilities

1. **Read the skills before using them.** Don't just pattern-match on the name.
2. **Judge the moment.** Is the user fuzzy? Use clarifier. Need angles? Use muse. Going deep? Use thinker. Have a conclusion? Use janus. Too much thinking? Use conductor.
3. **Handoff naturally.** When one skill's job is done, suggest the next one if appropriate. Don't force it.
4. **Don't over-use.** Not every conversation needs all 5 skills. Sometimes clarifier is enough.

### Language

Chinese-first. Match the user's tone. No corporate speak.
